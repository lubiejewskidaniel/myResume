
# 📄 My Online Resume

Welcome to my personal resume website — a live, interactive CV built using HTML, CSS, and JavaScript.  
This project showcases my background, education, skills, and career goals in a clean, responsive format that’s easy to share with potential employers.

🌐 **Live Demo:** [View My Resume](https://lubiejewskidaniel.github.io/myResume/)

---

## 💡 Features

- 📱 Mobile-responsive layout
- 📄 Structured sections: About Me, Skills, Education, Contact
- 🎨 Custom CSS styling (no frameworks)
- ⚡ Lightweight and fast-loading
- 📂 Easy to clone and adapt for your own resume

---

## 🚀 Tech Stack

- **HTML5** – Semantic structure
- **CSS3** – Custom layout and styling
- **JavaScript** – (Optional interaction/future-ready)
- **GitHub Pages** – Hosting and deployment

---

## 📁 Project Structure

```bash
myResume/
```
